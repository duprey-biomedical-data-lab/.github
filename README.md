# Duprey Biomedical Data Lab

We are a research group at the University of Kentucky College of Pharmacy. Our work focuses on delirium and substance use disorders, with an emphasis on medication safety and outcomes in acutely ill patients.

## Why this organization exists

Every analysis we publish should be something another researcher can pick up, read, and reproduce. This organization is where we keep the code behind our projects and papers. When we submit a manuscript, the analysis code lives here so reviewers and readers can see exactly what we did.

## How our repositories are organized

- **One repository per project or paper.** Each analysis project gets its own repo, started from our [lab-analysis-template](https://github.com/duprey-biomedical-data-lab/lab-analysis-template).
- **Naming convention:** short and descriptive, all lowercase, hyphens between words. For example: `icu-delirium-sedation-outcomes` or `oud-treatment-retention`.
- **No data in repos, ever.** We work with protected health information. Code and documentation are versioned here. Data stays on approved, secure storage and never touches GitHub. See the template repo for details on how we handle this.
